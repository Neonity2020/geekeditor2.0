直到1618年，瑞士数学家雅各布·伯努利才发现了自然对数e的真正本质。伯努利发现，如果我们将複利公式

$A = P(1 + r/n)^{nt}$

```
A = P(1 + r/n)^{nt}
```

其中

* A 是本金加利息的总金额
* P 是本金
* r 是年利率
* n 是每年复利的次数
* t 是年数

取极限，当n趋向于无穷大时，我们就得到了$A = Pe^rt$

```4
A = Pe^rt
```

这个公式被称为连续复利公式，它表明，随着复利次数的增加，本金和利息的总额会以指数方式增长。e就是这个指数增长过程中的一个重要常数。
